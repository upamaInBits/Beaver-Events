
# Beaver Events Mobile App (React Native + Firebase)

A React Native mobile application for discovering and managing campus events.

## Features
- Create new events (admin)
- Browse upcoming events
- View event details
- Find friends to go to an event with
- Chat with friends
- User-friendly navigation and interface

## Tech Stack
- React Native
- JavaScript
- Android / iOS project structure
- Firebase / Google services configuration required for full functionality

## Project Notes
This repository does not include sensitive configuration files such as:

- `android/app/google-services.json`
- `ios/GoogleService-Info.plist`

These files were excluded for security reasons.  
If you want to run the app with Firebase/Google services enabled, you will need to add your own configuration files.

## Prerequisites
Before running this project, install:

- Node.js
- npm
- React Native CLI environment
- Android Studio (for Android emulator or device testing)

For iOS:
- Xcode
- a compatible macOS version
